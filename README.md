# ❤️ Heart Disease Prediction System

A Machine Learning web application that predicts the likelihood of heart disease based on medical parameters.

The model is trained using Python and Scikit-learn and deployed as a web application using Flask.

---

## 🚀 Features

* Predicts heart disease risk using a trained ML model
* Simple web interface for user input
* Built with Python and Flask
* Deployed online using Render

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Flask
* Gunicorn
* HTML / CSS

---

## 📊 Machine Learning Model

The model is trained on a heart disease dataset using supervised learning techniques.
It takes medical attributes such as:

* Age
* Sex
* Chest pain type
* Blood pressure
* Cholesterol level
* Maximum heart rate
* And other health indicators

The model predicts whether a patient is at risk of heart disease.

## ⚙️ Installation (Run Locally)

Clone the repository:

```
git clone https://github.com/amolsingh05/heart-disease-ml.git
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
